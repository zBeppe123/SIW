Sezione Amministratore

Benvenuto, qui sei nella sezione Amministratore del sistema.
Per fare una modifica, registrazione o altro, scegli una voce tra questi elencati:

    Registra:

  • Registra un nuovo torneo.
  • Registra un nuovo arbitro.
  • Registra una partita di un torneo.

    Modifica:

  • Modifica i dati di un torneo.
  • Modifca i dati di un arbitro.

    Cancella:

  • cancella un torneo dal database.
  • cancella un arbitro dal database.
  • cancella una partita dal database.
